  <b>ColorLab: 专业色彩分析工具</b>
</h1>

<div align="center">

[ColorLab](https://colorlab.antv.vision) 是一个专业的色彩实验室，为设计师、开发者和数据分析师提供多样化的色彩分析和生产工具。

色彩是产品前端设计中的关键战略资产，直接影响用户感知和业务成果。

- <ins>构建品牌识别与信任</ins>：一致的色彩方案可将品牌识别度提升高达 80%，培养用户的熟悉感和可信度。
- <ins>引导用户行为与可用性</ins>：策略性的色彩对比可将注意力引导至关键操作（如主要按钮），改善导航和任务完成率。
- <ins>唤起情感与塑造感知</ins>：色彩触发特定的心理反应，使设计师能够创造期望的体验——从平静（蓝色）到紧迫（红色）。
- <ins>增强无障碍性与包容性</ins>：适当的色彩对比确保视觉障碍用户能够感知信息，扩大市场覆盖范围并符合法律标准。

`ColorLab` 旨在帮助设计师和开发者创建美观且有效的调色板，是 [AntV](https://github.com/antvis) 项目家族的一部分。

## ✨ 功能特性

- **专业色彩分析**：提供色彩预览、专业分析测试不同功能模式，为不同用户提供不同级别的功能。包括专业分析测试，如距离矩阵、色彩模型等创新色彩分析能力。
- **色彩优化**：针对色彩空间距离、色盲测试结果进行优化，提供一键优化能力，通过专业工具优化您的色彩。
- **导入导出**：提供一键导出和一键导入功能，让用户能够轻松快速生成和导入色彩。还支持从图片分析主题调色板并优化导出。

还有更多色彩工具，欢迎[探索](https://colorlab.antv.vision)。

使用以下命令在本地启动网站：

```bash
# 克隆仓库
$ git clone git@github.com:antvis/ColorLab.git

# 进入项目目录
$ cd ColorLab

# 安装依赖
$ npm install

# 启动本地开发服务器
$ npm run dev
```

然后您可以在 [http://localhost:8000](http://localhost:8000) 本地运行项目。
